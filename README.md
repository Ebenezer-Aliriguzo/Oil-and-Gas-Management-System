# Oil and Gas Management System  

This is a **React + Vite** web application for an Oil and Gas company. The system provides an easy-to-use platform for customers to learn about services, order LPG gas products, access safety tips, and contact support.  

The project demonstrates **modern web development** with React, Material-UI, and client-side routing.

---

## 🚀 Features  

- **Responsive Navigation Bar** with desktop and mobile views  
- **Search functionality** to quickly find pages (Home, Products, Services, etc.)  
- **Pages included**:  
  - Home  
  - LPG Gas Products  
  - Services  
  - Order Gas (with form)  
  - Safety Tips  
  - About Us  
  - Contact Us  
  - FAQs  
  - Login / Signup  
- **Mobile Drawer Menu** (collapsible side menu)  
- **Smooth navigation** between sections with scroll support  
- **Integrated branding** (Chidex Gas Plant Logo)  

---

## 🛠️ Tech Stack  

- **Frontend Framework**: [React](https://reactjs.org/) with [Vite](https://vitejs.dev/)  
- **UI Library**: [Material-UI (MUI)](https://mui.com/)  
- **Routing**: [React Router](https://reactrouter.com/)  
- **Deployment**: [Vercel](https://vercel.com/)
